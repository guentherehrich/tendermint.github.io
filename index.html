Tendermint

Open Blockchain Platform

The Tendermint Blockchain Platform removes all of the complexity of blockchain application development, allowing application developers to focus on business logic.

Tendermint is an open blockchain platform.

Tendermint can support all blockchain applications, including Bitcoin, Ethereum, and ErisDB. Developers building on Tendermint can also leverage the open-source ecosystem to build entirely new applications, in any programming language.

Tendermint is fast, secure, and scalable.

Built from the ground up for the needs of enterprise clients, the Tendermint blockchain engine supports applications with the best in speed, security, & scalability

Tendermint is for rapid ideation

Tendermint accelerates blockchain development by solving the common concerns including BFT consensus, P2P networking, transaction broadcasting, block storage, and more. We also provide open-source libraries for rapid blockchain application development.
